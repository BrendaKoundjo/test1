git config: pour la configuration permet de creer une configuration pour tous mes depot ou a chaque depot : c'est la notion de visibilite
.pour une configuration globale git config --global
.pour une configuration local 
 -git config user.name "christ"
 -git config user.email "christmagne0310@gmail.com"
 -git config color.diff auto: pour definir la couleur
.pour voir toutes les configuration possible nous pouvond tapper la commande git congit --list

comment faire suivre notre fichier par git: 
.pour avoir la statue: git status
.pour faire le commit je dois d'abord l'indexer
 git add texte.txt
.ensuite je peut faire le commit
git commit -m "creation de texte"

il existe une commande qui fait les deux a la fois
.git commit -a -m "commentaire"

pour avoir l'historique de mes commits
git log
pour renommer un fichier 
.git mv nomactuel.txt nomnouveau.txt
ensuite on fait le commit
